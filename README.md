# Инструкция: размещение SEO-страниц на GitHub Pages

## Что у вас есть:

5 готовых HTML-страниц:
- `index.html` — главная страница со списком статей
- `mrt-zaklyuchenie-neirohirurg.html` — про МРТ и нейрохирурга
- `bol-poyasnitsa-protruziya.html` — про боль в пояснице
- `golovokruzhenie-osteohondroz-shei.html` — про головокружение
- `aritmiya-srdechnogo-ritma.html` — про аритмию

## Шаг 1: Создать аккаунт на GitHub (если нет)

1. Перейдите на https://github.com
2. Нажмите "Sign up" (Регистрация)
3. Введите email, пароль, имя пользователя
4. Подтвердите email
5. Выберите бесплатный план (Free)

## Шаг 2: Создать репозиторий

1. Нажмите зелёную кнопку "New" (или "+" → "New repository")
2. Введите имя: `medical-advice-info` (или любое другое)
3. Важно: поставьте галочку "Public" (публичный)
4. Нажмите "Create repository"

## Шаг 3: Загрузить файлы

### Вариант А: Через браузер (проще)

1. В репозитории нажмите "Add file" → "Upload files"
2. Перетащите все 5 HTML-файлов
3. Внизу напишите комментарий: "Initial commit"
4. Нажмите "Commit changes"

### Вариант Б: Через консоль (если умеете)

```bash
git clone https://github.com/ВАШ_НИК/medical-advice-info.git
cd medical-advice-info
cp /путь/к/файлам/*.html .
git add .
git commit -m "Initial commit"
git push origin main
```

## Шаг 4: Включить GitHub Pages

1. В репозитории перейдите в "Settings" (шестерёнка сверху)
2. В левом меню выберите "Pages"
3. В разделе "Source" выберите "Deploy from a branch"
4. В выпадающем списке выберите "main" (или "master")
5. Нажмите "Save"
6. Подождите 2-5 минут

## Шаг 5: Проверить сайт

Ваш сайт будет доступен по адресу:
```
https://ВАШ_НИК.github.io/medical-advice-info/
```

Например:
```
https://demidchik.github.io/medical-advice-info/
```

## Шаг 6: Проверить индексацию (через 1-2 недели)

1. Откройте Google
2. Введите: `site:ВАШ_НИК.github.io/medical-advice-info`
3. Если страницы появились — Google их проиндексировал!

## Как обновлять контент:

1. В репозитории нажмите на файл
2. Нажмите карандаш (Edit)
3. Внесите изменения
4. Нажмите "Commit changes"
5. Изменения появятся на сайте через 1-2 минуты

## Добавить больше страниц:

1. Создайте новый HTML-файл
2. Загрузите в репозиторий (как в Шаге 3)
3. Добавьте ссылку на него в `index.html`

## Советы по SEO:

1. **Мета-теги** — в каждом файле уже прописаны title и description для поисковиков
2. **Ключевые слова** — в статьях естественно вписаны фразы, по которым ищут
3. **Ссылки** — с главной страницы ведут на статьи (это важно для индексации)
4. **Контент** — полезный, решает конкретные проблемы пациентов

## Что делать дальше:

1. Разместите ссылку на сайт в соцсетях, форумах
2. Добавьте в описание Telegram-канала
3. Со временем добавляйте новые статьи (Google любит свежий контент)
4. Через месяц проверьте статистику: https://ВАШ_НИК.github.io/medical-advice-info/

## Проблемы и решения:

**Сайт не открывается:**
- Проверьте, что репозиторий Public (не Private)
- Подождите 10 минут после включения Pages
- Проверьте, что файл называется `index.html` (с маленькой буквы)

**Страницы не индексируются:**
- Зарегистрируйтесь в Google Search Console: https://search.google.com/search-console
- Добавьте свой сайт
- Отправьте на индексацию главную страницу

**Нужна помощь:**
- GitHub Docs: https://docs.github.com/en/pages
